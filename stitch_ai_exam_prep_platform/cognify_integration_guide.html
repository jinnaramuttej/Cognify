# Cognify Integration Guide ## 1. Global Setup To match the "Definitive Final" Stitch designs exactly, ensure your Next.js project is configured as follows: ### Fonts (Newsreader + Inter) ```tsx // app/layout.tsx import { Inter, Newsreader } from 'next/font/google' const inter = Inter({ subsets: ['latin'], variable: '--font-inter' }) const newsreader = Newsreader({ subsets: ['latin'], variable: '--font-newsreader' }) export default function RootLayout({ children }) { return ( {children} ) } ``` ### Tailwind Config (Design Tokens) ```ts // tailwind.config.ts theme: { extend: { colors: { background: '#0F0F0F', card: '#1E1E1E', 'card-hover': '#252525', primary: '#EDE8DC', 'on-primary': '#0F0F0F', secondary: '#8A8A8A', }, fontFamily: { headline: ['var(--font-newsreader)', 'serif'], body: ['var(--font-inter)', 'sans-serif'], } } } ``` ## 2.
